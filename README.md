# Digital Electronics & VLSI Portfolio

![Language](https://img.shields.io/badge/Language-VHDL-blue)
![Simulator](https://img.shields.io/badge/Simulator-GHDL-green)
![Platform](https://img.shields.io/badge/Simulation-GTKWave-orange)

Digital hardware design projects developed as part of my **Digital Electronics & VLSI Systems Internship**. This repository documents my journey of learning digital system design through hands-on implementation of fundamental hardware components and computer architecture concepts using VHDL.

The projects progress from communication protocols to processor design, emphasizing modular RTL development, finite state machine (FSM) design, hardware verification, and simulation-based validation using **GHDL** and **GTKWave**. Each project is independently documented with architecture diagrams, simulation waveforms, and detailed implementation notes to showcase both the design methodology and the underlying digital electronics concepts.
